# BrainFuck
Weird programming language. 
Scripts:
- [x] A-Z -> ALL-LETTERS.b
- [x] a-z -> all-letters.b
- [x] 0-9 -> all-numbers.b

## Convert to ASCII
Convert your characters to ASCII. Brainfuck prints characters based on their ASCII value.
For example, it will print the letter A if it detects the value 65.
